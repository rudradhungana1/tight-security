----------Registration and Login Prototype-----------
A login and registration form using HTML, PHP,JavaScript and MySQL. This form allows users to register and login. 
All information is stored in a MySQL database. After successful login the user is redirected to their dashboard.
I enjoy making it so much! 

Features
===============
* Stores passwords in DB as hashes
* Modern and minimalistic design
* Easy logout via logout.php
* Password strength
* Email and Username verification
* Google reCAPTCHA
* Login timeout
* designed for security

Steps to Run programme
======================
> Open Xampp mysql database.
> Load the sql file in phpmyadmin with DB details given in cybersecurity.sql
> Open the folder in localhost "http://localhost/CyberSecurity/login.php" .
